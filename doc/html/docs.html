Live Looping Sampler

Documentation

The documentation is divided into the following sections:

  • Getting Started discusses installation and application startup.
  • Command Reference contains detailed description of each command available for realtime control of the looping.
  • MIDI Bindings shows how to set up appropriate MIDI bindings for your controller.
  • Sync/Tempo describes how the tempo and synchronization parameters allow you to sync your loops to external forces or other loops.
  • OSC Interface describes the low-level OSC interface to the looping engine.
